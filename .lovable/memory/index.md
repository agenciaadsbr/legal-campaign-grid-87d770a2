## Core
Dark mode = azul escuro profundo em TUDO (background ~222 47% 8%, cards ~222 45% 11%, sidebar ~222 47% 10%, bordas ~220 25% 18%). Nunca cinza ou preto puro.
Light mode = TUDO claro, inclusive sidebar (sidebar-background 0 0% 100%, foreground 222 40% 12%). Sem azul escuro na sidebar no tema claro.
Sempre usar tokens semânticos do index.css (bg-background, text-foreground, bg-card, bg-sidebar, border-border). Nunca cores hex/rgb hardcoded em componentes.
Todas as cores em HSL no index.css e tailwind.config.ts.
